A short piece of code that shows how pyTorch is able to generate an image by combining a Stable Diffusion Pipeline and the Python Imaging Library

pyTorch is able to generate an image using a string prompt.
CLIP text model is used for text encoding
Stable Diffusion Pipeline is used to generate images
PIL is a library used for image processing (Python Import Library)

note:
'pip3 install accelerate' installs a package fundamental for pytorch, takes a lot of memory nad would require time to download and install.
Accelerate is a library used to write minimal torch code without boilerplate code.
